# LeadDesk Next.js Website

A production-ready Next.js 14 landing page for LeadDesk, built with App Router, TypeScript, Tailwind CSS, Framer Motion, SEO metadata, JSON-LD structured data, sitemap, robots route, and a strict black/white/grayscale visual system.

## Quick Start

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

## Production

```bash
npm run build
npm start
```

## Environment

Copy `.env.example` and set:

```bash
NEXT_PUBLIC_SITE_URL=https://your-domain.com
NEXT_PUBLIC_CALENDLY_URL=https://calendly.com/your-team/demo
NEXT_PUBLIC_CONTACT_EMAIL=hello@your-domain.com
```

## SEO Included

- Metadata API title, description, canonical URL, Open Graph, and Twitter Card tags
- Schema.org JSON-LD for Organization, Product, FAQPage, and BreadcrumbList
- App Router sitemap and robots routes
- Semantic headings and landmark structure
- Keyword coverage for lead generation, sales automation platform, CRM software, lead scraping tool, email outreach platform, B2B lead generation, cold outreach software, and lead management system

## Design Notes

The UI intentionally uses only black, white, and grayscale values. Interactive states, shadows, borders, icons, assets, and focus styles stay within the grayscale palette.
